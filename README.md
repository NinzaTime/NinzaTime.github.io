## Deployment

Hosted on GitHub Pages: [https://ninzatime.github.io/](https://ninzatime.github.io/)
## Structure

- `index.html` – Homepage
- `reflection.html` – Full course reflection
- `project.html` – Project 2: Global Revision (major project)
- `minor.html` – Minor writing revisions
- `style.css` – Global styles
- `script.js` – Shared JavaScript (theme toggle, typewriter, reveal)

## Features

- Light/Dark mode toggle  
- Typewriter effect on homepage  
- Reveal animations for sections and cards  
- Bento-grid layout for minor writings  
- Highlighted revisions
